## Why DopaLin?

Existing Linux monitoring tools are either outdated or lack features. DopaLin provides a modern, beautiful interface with powerful features for gamers and power users.

## Planned Features

- Real-time hardware monitoring (CPU, GPU, RAM, temperatures)
- CPU governor control (Performance/Balanced/Power Save)
- Gaming tools (ProtonDB/SteamDB integration)
- vkBasalt GUI configuration
- System tweaks and optimizations
- beautiful themes
